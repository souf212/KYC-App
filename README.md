# KYC (Know Your Customer) Application

This repository contains the KYC (Know Your Customer) Application, consisting of a front-end mobile application and a back-end service. This system is designed for digital counter terminals with dual touch displays, facilitating a modern, banking-grade onboarding flow.

## Project Structure

- **`kyc/`**: The front-end mobile application built with React Native and Expo. It provides the user interface for step-by-step guidance, document capture, face capture, and signature capture.
- **`kyc-backend/`**: The back-end service built with .NET. It handles the API requests, business logic, and potentially data persistence for the KYC process.
- **`doc/`**: Documentation files related to the project.

## Features

- **Step-by-Step Onboarding**: Clear guidance system for users.
- **Document & Face Capture**: Optimized experience for capturing required KYC documents and user photos.
- **Signature Module**: Banking-grade signature capture.
- **Dual-Screen Support**: Optimized UI for 10.1-inch touchscreens and dual-screen terminals.
- **Agent Dashboard**: Interface for agents to monitor and assist with the onboarding process.

## Getting Started

### Front-End (kyc)
1. Navigate to the `kyc` directory.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the Expo development server.

### Back-End (kyc-backend)
1. Navigate to the `kyc-backend` directory.
2. Open the solution file (`.slnx`) in Visual Studio or your preferred IDE.
3. Build and run the project.
